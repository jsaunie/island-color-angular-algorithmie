The goal of this exercise is to discover all islands in the map and color them with a unique color

An island is defined by all connecting Land squares to each other

An island can have a size of 1

Each square is connected to 8 others squares

1. Color all islands

2. Count the number of islands

3. To go further, you can add an event listener to the canvas element and change the color of the island to the color chosen in the text field,

You should not change the existing code, and only write the necessary code

Bonus: test your code, personalize css
